# Methods alignment note

This GitHub-ready package was organized to mirror the study methods:
- baseline chronic conditions harmonized in UKB and CKB,
- aortic disease as a composite endpoint,
- Cox regression for 42 chronic conditions,
- multimorbidity analysis,
- PAF analysis in UKB,
- sex-stratified analysis, and
- competing-risk sensitivity analysis.

The uploaded code supplied for this repository preparation covered the UKB side of the workflow. The CKB side remains described in the manuscript Methods and should be added to the repository separately if code-sharing for CKB is also required.
