# Choose a license before uploading

This package is ready for GitHub structurally, but you should choose the final license yourself before public release.

Common choices:
- MIT: simple and permissive
- BSD-3-Clause: permissive with attribution conditions
- GPL-3.0: stronger copyleft

Do not publish without checking:
1. your institution's policy,
2. funder requirements,
3. consortium or biobank agreements, and
4. whether co-authors agree on the licensing choice.
