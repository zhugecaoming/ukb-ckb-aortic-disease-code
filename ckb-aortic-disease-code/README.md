# CKB aortic disease analysis code

This repository contains the analysis code for a study of multimorbidity and risk of aortic diseases using data from the China Kadoorie Biobank (CKB).

## Repository structure

- `code/`: analysis scripts and utility functions
- `data-raw/`: placeholder only; raw CKB data are **not** included
- `results/`: output tables generated by the scripts
- `docs/`: supporting documentation
- `run_all.R`: optional wrapper to run the full workflow

## Data availability

The individual-level data used in this study come from the China Kadoorie Biobank (CKB) and are **not publicly available** in this repository because of data access restrictions. Researchers who meet the criteria for access should apply through the official CKB data access procedures.

This repository is therefore intended to share:

- analysis code
- outcome and exposure construction logic
- statistical modeling workflow
- output table generation workflow

It does **not** include:

- raw participant-level data
- derived participant-level analytic datasets
- any files that would violate CKB data access rules

## Expected input files

Place the following files in `data-raw/` before running the scripts:

- `baseline_data.RData`
- `events_data.RData`

These files should contain objects named:

- `baseline_data`
- `events_data`

## Analysis workflow

Run the scripts in the following order:

1. `code/00_packages.R`
2. `code/01_prepare_events_and_exposures.R`
3. `code/02_define_aortic_outcomes.R`
4. `code/03_merge_total_dataset.R`
5. `code/04_cox_analysis.R`
6. `code/05_multimorbidity_analysis.R`
7. `code/06_paf_analysis.R`
8. `code/07_baseline_characteristics.R`

Or run everything with:

```r
source("run_all.R")
```
