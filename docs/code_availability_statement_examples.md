## Data availability statement (recommended)

> UK Biobank data used in this study are available from UK Biobank upon application and approval (Application ID: 106528). China Kadoorie Biobank data are available through the CKB data access process subject to review and approval by the CKB investigators and governing policies. These controlled-access human participant data are therefore not publicly distributed through this GitHub repository.
