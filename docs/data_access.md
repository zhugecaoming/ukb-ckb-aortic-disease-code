# Data access and sharing statement support

## What is shared in this repository

This repository shares:
- analysis scripts
- configuration templates
- documentation describing inputs and outputs
- figure/table generation workflows, where applicable

## What is not shared

This repository does **not** include:
- raw UK Biobank participant-level data
- raw China Kadoorie Biobank participant-level data
- controlled-access files that cannot be redistributed under the relevant data-use agreements
- identifying participant information

## Why the data are not included

The study uses human participant data obtained under controlled-access or licensed data-use conditions. These data cannot be redistributed openly through GitHub.

## Data-access wording for manuscript use

### Data availability statement draft

> UK Biobank data used in this study are available from UK Biobank upon application and approval (Application ID: 106528). China Kadoorie Biobank data are available through the CKB data access process subject to review and approval by the CKB investigators and governing policies. The present GitHub repository does not contain raw participant-level data because these datasets are subject to controlled-access restrictions designed to protect participant privacy and to comply with the terms of use of the source cohorts.

### Code availability statement draft

> The code used for data processing, statistical analyses, Mendelian randomization analyses, and machine-learning model development is available in the public GitHub repository at: https://github.com/zhugecaoming/UKB--CKB-poject . An archived release with persistent identifier should be cited in the final published version when available.
